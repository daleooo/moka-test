# Component Hierarchy

- Sidebar

    - JobRow
    - JobRow
  - JobBox

## To use

```sh
npm install
node server.js
```
And visit <http://localhost:3000/>.

## Changing the port

You can change the port number by setting the `$PORT` environment variable before invoking any of the scripts above, e.g.,

```sh
PORT=3001 node server.js
```
